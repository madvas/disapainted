Disapainted.com is a web application for creating pivot animations online. Pivot animations are funny 2D stick-man animations, which you easily create by dragging stick figure handles. This web app is heavily inspired by Pivot Animator desktop application.

Interestingly, Disapainted is completely open source and you can find its source code on GitHub. Anyone is more than welcome to contribute!

I'd also like to express infinite thanks to following open source technologies, which were used to build Disapainted
